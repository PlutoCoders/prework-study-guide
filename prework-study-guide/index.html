HTML

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The head element contains information about the webpage, such as metadata.
  • The body element represents the visible content shown to the user.
  • Metadata examples for head: title, style, base, link, meta, script.
  • p is for your paragraphs, ui for unordered lists, ol for ordered lists, li for list items.
  • img for images, src (attribute) for defined location of image, alt for explaining the image.
  • br is a line break to create space between two blocks of code.
  • a href=link where a is for an anchor which creates links, href is the url for the link.
  • header appears at the top and usually contains navigational/introductory content.
  • h1 to h6 are headings, to be used in order, where h1 is used once per page, and the others are used after. (Each header gets smaller.)
  • footer at the bottom of the page, contains navigation, copyright, sitemap, contact, etc. In our case we have a note.
  • Learn more about HTML/Links!

CSS

  • A margin indicates how much space we want around the outside of an element.
  • the most common way to style a webpage is by using an external CSS style sheet. This is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier. When using an external style sheet, we link the file using a <link> element placed in the HTML's <head>, as shown in the following code: <head> < rel="stylesheet" href="./assets/style.css" /> </head>
  • margin: 40px auto; (use ; to add more stlying), assigns 40px to the top and bottom, and the auto value to the left and right.
  • A padding indicates how much space we want around the content inside an element. ex/ padding: 10px 50x;
  • A period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS.
  • width: 80% = Width of the element at 80% of the available width
  • border: 5px solid gray = assigns a border with a width of 5px that is solid and gray.
  • selector (property; value)
  • Sometimes, we want to apply a rule to all the elements visible on the page. To do that, we use a special wild card symbol, or *.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property. For example, the declaration color: blue; includes the CSS property color and the value for that property, blue.
  • Learn more about CSS styling!
